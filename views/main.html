What's with this ?

APOJOP is an object pretty-printer for AngularJS. Basically, it does what you'd do as a developer : it makes your JavaScript objects look good in HTML. You don't want your objects to be printed one key/value per line, but the basic JSON.stringify(myObject, undefined, 2) is not enough for you. You just want something that prints objects on one line where they're not too long, and unfold them when they are. Well, here is APOJOP.

Output

The printer below will try to print lines which are shorter than {{max_columns}} character{{(max_columns > 1 && "s" || "")}}. If a line is longer than that and contains complex things such as objects or arrays, it will unfold it. Otherwise, it will simply print it. Try to change the treshold below !

The printer below will try to unfold {{max_depth}} level{{(max_depth > 1 && "s" || "")}}. Sometimes unfolding is not possible, for instance with primitive objects such as dates, numbers, strings... Then the printer will simply print the value, regardless of the desired depth setting.


  

  

    

  




Code



HTML

 <pre ng-bind-html="myObject | pretty:max_columns:'columns'"></pre>
 <pre ng-bind-html="myObject | pretty:max_levels:'levels'"></pre>
